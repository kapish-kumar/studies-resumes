Programação — Ano 1

1. Noções Básicas de C++

História e Objetivo:

Desenvolvido por Bjarne Stroustrup (1983)

Extensão da linguagem C com programação orientada a objetos (OOP)

Usado em software de sistema, desenvolvimento de jogos, programação competitiva, etc.

Estrutura de um Programa em C++:

#include <iostream>
using namespace std;
int main() {
  cout << "Olá, Mundo!";
  return 0;
}

Comentários:

Linha única: // Este é um comentário

Múltiplas linhas: /* Este é um comentário */

2. Tipos de Dados e Variáveis

Tipos Primitivos: int, float, double, char, bool

Modificadores: short, long, unsigned, signed

Declaração de Variáveis: int idade = 25; float pi = 3.14; char nota = 'A';

Constantes: const int MAX = 100;

3. Operadores

Aritméticos: + - * / %

Relacionais: == != > < >= <=

Lógicos: && || !

Bit a bit: & | ^ ~ << >>

Atribuição: = += -= *= /= %=

Incremento/Decremento: ++, --

4. Estruturas de Controlo

Decisão: if(condição) { } else if(condição) { } else { }

Switch case: switch(expressão) { case 1: break; case 2: break; default: break; }

Loops: for, while, do-while

Comandos de salto: break, continue, return, goto

5. Funções

Definição: int soma(int a, int b) { return a + b; }

Tipos: com/sem valor de retorno, com/sem parâmetros

Outros: Sobrecarga de funções, Funções inline, Recursão

6. Arrays e Strings

Array: int numeros[5] = {1,2,3,4,5};

Array Multidimensional: int matriz[2][3] = {{1,2,3},{4,5,6}};

Strings: estilo C: char str[] = "Olá"; C++ string: string str = "Olá";

7. Ponteiros

Definição: Armazena endereços de memória

Exemplo: int a = 10; int *p = &a;

Operações: *, &, ->

Memória dinâmica: new e delete

8. Referências

Nome alternativo para uma variável

Exemplo: int a = 10; int &ref = a;

9. Programação Orientada a Objetos (OOP)

Classes e Objetos:

class Carro { public: string marca; void conduzir() { cout << "A conduzir"; } };
Carro meuCarro; meuCarro.conduzir();

Encapsulamento: private, public, protected

Construtores e Destrutores

Herança: class Derivada : public Base {}

Polimorfismo: Tempo de compilação (Sobrecarga de funções e operadores), Tempo de execução (Funções virtuais)

Abstração: Classes abstratas e funções virtuais puras

10. Funcionalidades Avançadas

Sobrecarga de operadores: Complex operador + (const Complex &c);

Templates: Templates de funções e de classes

STL: Containers (vector, list, map, set, etc.), Iteradores e Algoritmos

Tratamento de Excepções: try { } catch(exception &e) { }

Ficheiros (File I/O): #include <fstream> ofstream out("ficheiro.txt"); ifstream in("ficheiro.txt");

11. Outros

Namespace: using namespace std;

Conversão de tipos: static_cast, dynamic_cast, reinterpret_cast, const_cast

Diretivas do pré-processador: #define, #include, #ifdef, #ifndef

Funções lambda

Ponteiros inteligentes: unique_ptr, shared_ptr

Multithreading: #include <thread>
